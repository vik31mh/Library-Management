About This Project

Welcome to the Library Management System, a mini-project developed by Navneeth and Vikas, students of the 5th Semester. This system is designed to simplify and enhance the management of library resources, making it easier for users to search, borrow, and manage books efficiently.

Key Features

  • Book Search and Availability: Search for books by title, author, or genre, and check availability.
  • AI Chatbot Integration: An AI-powered chatbot helps users with library-related queries.
  • User Management: Maintain a database of members and track borrowing history.
  • Responsive Design: Ensures usability across various devices.

Technology Stack

  • Frontend: HTML, CSS
  • Backend: Python, Flask
  • Database: SQL
  • AI Integration: OpenAI GPT for chatbot functionality

Our Journey

This project was developed as part of our 5th Semester Mini-Project. Through this journey, we:

  • Enhanced our skills in web development and database management.
  • Gained experience in integrating AI-powered tools into practical applications.
  • Worked collaboratively to design, implement, and test the system.

Future Enhancements

  • Adding an admin dashboard for better management.
  • Implementing SMS/email notifications for overdue books.
  • Integrating advanced AI features like personalized book recommendations.
